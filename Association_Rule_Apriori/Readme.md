* Code: Code.py

==[Input Data 說明]==
* 檔名: InputData.txt
    * Each line (row) in file is a transaction (每一列是一筆交易紀錄)
    * Each number is a item (每列中的每一個數字代表不同商品的編號，以空格區分)
    * Ex. Line 2: 30 31 32 表示第二筆交易紀錄中購買了 No. 30、No. 31、No. 32 的商品
